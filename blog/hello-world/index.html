Hello, World — pierwszy wpis na blogu

2026-04-01 PepeCodes 1 min read
python zsbg test

Witaj na blogu. To jest pierwszy wpis wygenerowany przez zsbg — statyczny generator stron oparty na Jinja2 i Markdownie.

Bloki kodu

Generator używa Pygments do kolorowania składni. Wystarczy oznaczyć blok trzema backtickamy i podać język.

Python

from zsbg.generator import Generator
from pathlib import Path

Generator(
    content_dir=Path("content"),
    output_dir=Path("output"),
    templates_dir=Path("templates"),
    pygments_style="monokai",
).build()

SQL

SELECT
    p.name,
    COUNT(o.id)        AS order_count,
    SUM(o.total_price) AS revenue
FROM products p
LEFT JOIN orders o ON o.product_id = p.id
WHERE p.owner_id = 42
GROUP BY p.id, p.name
ORDER BY revenue DESC NULLS LAST;

Bash

# Stwórz nowy wpis
zsbg new moj-nowy-wpis

# Zbuduj bloga
zsbg build --pygments-style github-dark

Formatowanie tekstu

Możesz używać pogrubienia, kursywy oraz kodu inline bezpośrednio w tekście. Linki wyglądają tak: Pygments styles.

Listy

Nieuporządkowana:

  • Jinja2 — silnik szablonów
  • python-frontmatter — parsowanie metadanych YAML
  • Markdown — konwersja do HTML
  • Pygments — kolorowanie składni
  • Click — interfejs CLI
  • Pydantic v2 — walidacja modeli danych

Uporządkowana:

  1. Napisz wpis w Markdownie
  2. Uruchom zsbg build
  3. Wdróż katalog output/ na serwer
